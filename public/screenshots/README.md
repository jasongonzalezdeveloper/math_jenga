# Screenshots Directory

This directory contains screenshots and visual assets for the README.

## Recommended Images

Add the following screenshots to showcase your Math Jenga project:

1. **tower-view.png** - Main view of the Jenga tower with numbered cubes
2. **question-modal.png** - Screenshot of the math question modal
3. **gameplay.gif** - Animated GIF showing the gameplay in action

## Image Guidelines

- **Format**: PNG for static images, GIF for animations
- **Size**: Recommended width 800-1200px for optimal display
- **Compression**: Optimize images to keep file sizes under 500KB each
- **Quality**: Use high-quality screenshots with clear, readable text

## Taking Screenshots

### For Tower View
- Show the full Jenga tower with visible numbered cubes
- Include the lives counter and rotation button
- Use a clean, uncluttered view

### For Question Modal
- Capture the modal with a sample math question
- Include the input field and buttons
- Show the clean UI design

### For Gameplay GIF
- Record a short sequence (5-10 seconds)
- Show: clicking a cube → solving the problem → placing it on top
- Keep file size under 3MB for fast loading
